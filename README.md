# libra

Minimalist book manager for organizing your home library.

# Komentarz

Projekt jest w fazie początkowej. Wymaga rozbudowy o nowe funkcjonalności. W szczególności należy dodać:
- aplikację webową do zarządzania książkami,
- refaktoryzację kodu,
- testy jednostkowe,
- security,
- odpowiednią architekturę,
- dokumentację.
